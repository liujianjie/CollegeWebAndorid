1、SQLServer 2000 与SQLServer 2005的区别？

SQL Server 2005包含了多项新功能，这使其成为大规模联机事务处理（OLTP）、数据仓库和电子商务应用程序的优秀数据库平台。

2、什么是E-R图，建立E-R图的依据是什么？为什么要绘制E-R图？

E-R图也称为实体－关系模型，由实体、属性、关系三个要素构成。
利用E-R图实现概念结构设计的方法叫做E-R方法,而概念设计是数据库设计的第一个阶段

3、什么是1NF?2NF?3NF?

1NF：第一范式 每个属性值都是由不可分割的最小数据单位组成
2NF：第而范式 没有字段依赖其它字段
3NF：第三范式 A>B>C A不能决定B B不能决定C
